## Influx
![GitHub contributors](https://img.shields.io/github/contributors/TeamInflux/Influx.svg)<br>

## Influx Introduction
-------------
**Influx is the advanced server software based on PocketMine-MP and Genisys! <br>
Influx supports packet and event debug system (for plugin developers) <br>

## Notice

**Influx is supported to Minecraft:PE 1.0.8 (PE & Win10) <br>
Report any issues and we will try to fix them for you.**  <br>

## Information

- This is Beta software, you will experience bugs and crashes.
- You can make a pull-request if you want to help.
- We currently are accepting helpers
